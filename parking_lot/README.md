# Parking Lot System Design (Java)

A menu-driven Parking Lot Management System built in Java following Object-Oriented Programming (OOP) principles and Low-Level Design (LLD) concepts.

## Features

- Park Bikes, Cars, and Trucks
- Remove Vehicles
- Automatic Ticket Generation
- Parking Fee Calculation
- Multiple Parking Spot Types
- Strategy Pattern for Parking Spot Allocation
- Menu-driven Console Application

## Tech Stack

- Java
- OOP
- Java Collections Framework

## Concepts Used

- Encapsulation
- Abstraction
- Inheritance
- Polymorphism
- Strategy Design Pattern

## Project Structure

```text
src
├── enums
├── model
├── service
└── App.java
```
